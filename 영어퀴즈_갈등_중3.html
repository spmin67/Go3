영어 단어 학습 & 퀴즈

주제: 갈등 | 수준: 중3

핵심 단어 10개

conflict - 갈등, 충돌
The conflict between the two groups lasted for many years.
→ 갈등은 / 두 그룹 사이의 / 지속되었다 / 여러 해 동안.
dispute - 논쟁, 분쟁
They had a dispute about who should lead the team.
→ 그들은 / 가졌다 / 논쟁을 / 누가 팀을 이끌어야 하는지에 대해.
argument - 언쟁, 논쟁
The argument between friends ended peacefully.
→ 언쟁은 / 친구들 사이의 / 끝났다 / 평화롭게.
disagreement - 의견 차이, 불일치
There was a disagreement about the final decision.
→ 있었다 / 의견 차이가 / 최종 결정에 대해.
tension - 긴장, 긴장 상태
The tension in the classroom grew during the test.
→ 긴장은 / 교실 안의 / 커졌다 / 시험 중에.
oppose - 반대하다
Many people oppose the new rule.
→ 많은 사람들이 / 반대한다 / 새로운 규칙을.
resolve - 해결하다
We need to resolve this problem quickly.
→ 우리는 / 필요하다 / 해결할 / 이 문제를 / 빨리.
negotiate - 협상하다
Both sides agreed to negotiate peacefully.
→ 양측은 / 동의했다 / 협상하기로 / 평화롭게.
compromise - 타협, 타협하다
A compromise is necessary to end the conflict.
→ 타협은 / 필요하다 / 갈등을 끝내기 위해.
peaceful - 평화로운
They found a peaceful solution to the dispute.
→ 그들은 / 찾았다 / 평화로운 해결책을 / 분쟁에 대한.
